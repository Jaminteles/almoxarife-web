## 📜 Sumário Executivo

**O que é?**  
Sistema de gerenciamento de Almoxarifado com funcionalidades de CRUD para Funcionários e Fornecedores.

**Tech Stack?**  
React 19 + Material-UI 7 + React Router 7

**Quanto falta?**  
~14-16 semanas para versão completa (com backend, testes, deploy)

**Como começo?**  
Execute `npm install && npm start`, depois leia [SETUP.md](./SETUP.md)

**Dúvidas?**  
Consulte [TROUBLESHOOTING.md](./TROUBLESHOOTING.md) ou pergunte ao time!
